# sfdp — Scalable Force‑Directed Layout Examples

Examples in this directory use Graphviz’s **sfdp** engine, a scalable force‑directed layout designed for large graphs.

Official Graphviz documentation:  
https://graphviz.org/docs/layouts/sfdp/

## What’s Here
- Self‑contained Relationship Visualizer workbooks  
- Sample data and SQL queries  
- Resulting Graphviz outputs (SVG/PNG)
