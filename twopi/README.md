# twopi — Radial Layout Examples

Examples in this directory use Graphviz’s **twopi** engine, which produces radial layouts with nodes arranged in concentric circles around a root.

Official Graphviz documentation:  
https://graphviz.org/docs/layouts/twopi/

## What’s Here
- Self‑contained Relationship Visualizer workbooks  
- Sample data and SQL queries  
- Resulting Graphviz outputs (SVG/PNG)  
- Brief notes explaining each example

## When to Use twopi
Use these examples when you need:
- Radial or circular hierarchy layouts  
- “Center‑outward” visualizations  
- Layered rings of related items  
- A clear visual root with expanding levels
