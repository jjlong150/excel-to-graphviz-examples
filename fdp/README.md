# fdp — Force‑Directed Layout Examples

Examples in this directory use Graphviz’s **fdp** engine, a force‑directed layout optimized for medium‑sized undirected graphs.

Official Graphviz documentation:  
https://graphviz.org/docs/layouts/fdp/

## What’s Here
- Self‑contained Relationship Visualizer workbooks  
- Sample data and SQL queries  
- Resulting Graphviz outputs (SVG/PNG)  
- Brief notes explaining each example

## When to Use fdp
Use these examples when you need:
- Natural, force‑directed layouts  
- Medium‑scale networks  
- Relationship maps without hierarchy  
- More organic spacing than neato
