# circo — Circular Layout Examples

Examples in this directory use Graphviz’s **circo** engine, which arranges nodes in circular or cyclic patterns.

Official Graphviz documentation:  
https://graphviz.org/docs/layouts/circo/

## What’s Here
- Self‑contained Relationship Visualizer workbooks  
- Sample data and SQL queries  
- Resulting Graphviz outputs (SVG/PNG)  
- Brief notes explaining each example

## When to Use circo
Use these examples when you need:
- Circular or ring‑based layouts  
- Cyclic or loop‑oriented structures  
- Relationship maps without a strict hierarchy  
- Even spacing around a central axis
